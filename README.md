用 Python 写一个纯文字的“赛博宠物/AI角色”状态模拟器。

必须使用 class 定义一个角色类（比如 class Dog: ）。

一，角色必须包含以下属性（初始化 __init__）：

	名字 (name)

	心情值 (mood: 0-100)

	体力值 (energy: 0-100)

二，角色必须包含以下行为（方法）：

	chat(): 聊天。体力消耗 10。如果心情 > 80，打印开心回复；如果 < 50，打印敷衍回复。

	sleep(): 睡觉。体力恢复 50。

	work(): 打工。心情下降 20，体力下降 30。

主程序逻辑： 写一个 while True 的循环，让用户在控制台输入指令（比如输入 1 聊天，2 睡觉，3 打工），调用角色的方法，并实时打印当前心情和体力。如果体力 < 0，提示累倒并退出程序。
